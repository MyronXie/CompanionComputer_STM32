## Command

Here is command for F3 board.

Cmd|Content
-|-
**0x00**|**System**
0x01|System Reset
0x02|USART Reset
-|-
**0x10**|**Battery Management**
0x11|Battery offboard
0x12|Vdiff > 100mV
0x13|Auto PowerOn Fail
0x14|FET Enable Fail
0x15|Battery Init Fail
0x16|Auto PowerOff Fail
-|-
**0x20**|**Landing Gear**
0x21|Landing Gear Auto Reset
